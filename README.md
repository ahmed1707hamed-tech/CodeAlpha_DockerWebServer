# 🚀 Docker DevOps Dashboard

## 📌 Overview

This project demonstrates a containerized DevOps dashboard using Docker and Nginx.

## 🛠️ Technologies

* Docker
* Nginx
* HTML, CSS, JavaScript

## 📂 Project Structure

* Dockerfile
* index.html

## 🚀 Run Locally

```bash
docker build -t ahmed-nginx .
docker run -d -p 8080:80 ahmed-nginx
```

Then open:
http://localhost:8080

## 🎯 Features

* DevOps Dashboard UI
* Dark mode toggle
* Live data simulation
